# Troubleshooting

## Missing files

Run the validator against the repository and the installed target.

## Broken install

Check `OPENCLAW_HOME`, `OPENCLAW_SECRETS_FILE`, and the target layout created by the bootstrapper.

## Permission errors

Confirm the install target is writable and the generated `bin` scripts are executable.

## Unexpected data

Any personal data, live logs, or feature-specific artifacts should stay out of the public core and into a private overlay.
