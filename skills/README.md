# Skills

This directory contains the public OpenClaw skill set.

Current coverage:

- Bootstrap and install
- Operations and maintenance
- Validation and release gating
- Recovery and observability
- Security and plugin boundaries

## Public skill set

- `openclaw-bootstrap`
- `openclaw-ops`
- `openclaw-security`
- `openclaw-plugins`
- `openclaw-validate`
- `openclaw-observability`
- `openclaw-release`
- `openclaw-recovery`

## Guidance

- Skills should explain when to use them, what they do, and what they must not touch.
- Skills should remain short enough to be readable, but complete enough to be actionable.
- Anything feature-specific or user-specific belongs in a private overlay or plugin repo.

Feature-specific automations belong in private overlays or separate plugin repositories.
